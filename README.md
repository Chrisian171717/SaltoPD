Descripcion del proyecto:
Esta empresa  se dedica a asistir al Departamento de Policía de Salto con una aplicación que facilita la detección del contrabando terrestre y facilita la realización de tareas más eficaces, como por ejemplo: la búsqueda de un civil, el código penal, las denuncias, los vehículos.

Misión
Nuestra misión es potenciar a las fuerzas de seguridad del Uruguay con una plataforma tecnológica integral que centralice, gestione y brinde acceso instantáneo a información crítica de civiles, vehículos y denuncias, facilitando la identificación, la toma de decisiones y la coordinación en tiempo real para una respuesta policial más eficiente y efectiva, contribuyendo a la seguridad y el orden público.

Funcion del proyecto:
Fortalecer la coordinación y cooperación entre las distintas unidades y niveles de las fuerzas de seguridad, promoviendo un accionar conjunto, eficiente y estratégico en la prevención y combate del contrabando y otras actividades delictivas.
Optimizar los procedimientos de control, registro y gestión de información policial, mediante la implementación de una herramienta tecnológica que incremente la eficiencia operativa y la precisión en la toma de decisiones.
Contribuir al fortalecimiento de la seguridad pública y nacional, garantizando un acceso ágil y seguro a información crítica, y potenciando la capacidad de respuesta inmediata ante incidentes y situaciones de riesgo.
Disminuir la incidencia del contrabando y delictividad en general, a través de un sistema integral que facilite la identificación rápida de personas y vehículos vinculados a actividades ilícitas, mejorando así la efectividad del accionar policial.

Utilizamos XAMPP como base de datos, ahi podemos guardar informacion de civiles, vehiculos, etc; con sus especificaciones (EJ: civiles (Nombre, Edad, etc))
Para HTML5, CSS, JS y PHP utilizamos Visual Studio Code.

¿Para que sirve cada cosa?
XAMPP (Base de datos):
Una base de datos sirve para almacenar, organizar y gestionar información de manera estructurada, de modo que sea fácil acceder, buscar, modificar y analizar esos datos cuando se necesiten.

HTML5:
HTML5 es el lenguaje base de toda página web.
Sirve para estructurar y organizar el contenido que se muestra en el navegador. Es como el “esqueleto” del sitio: define qué elementos hay y en qué orden aparecen (títulos, párrafos, imágenes, videos, botones, formularios, tablas, etc.).

CSS:
CSS se usa para dar estilo, diseño y presentación visual a las páginas creadas con HTML.
Con CSS podés cambiar colores, fuentes, tamaños, márgenes, bordes, posiciones, fondos, transiciones, animaciones, y adaptar el diseño a distintos dispositivos (celulares, tablets, computadoras).

Java Script (JS):
JavaScript es un lenguaje de programación que permite darle vida e interactividad a las páginas web.
Se ejecuta directamente en el navegador del usuario, y sirve para que los elementos reaccionen a lo que hace el visitante (por ejemplo, mostrar mensajes, validar formularios, cambiar contenidos sin recargar la página, abrir menús, crear animaciones o sliders, etc.).

PHP:
PHP es un lenguaje de programación que se ejecuta en el servidor, antes de que la página llegue al navegador del usuario.
Sirve para conectar la página web con bases de datos, manejar usuarios, procesar formularios, generar contenido personalizado, enviar correos, crear sistemas de login, etc.

Tambien utilizamos phyton.
Phyton:
Python es un lenguaje de programación de propósito general, es decir, se puede usar para casi cualquier tipo de desarrollo.
Se destaca por tener una sintaxis simple y fácil de leer, parecida al lenguaje humano, lo que lo hace ideal tanto para principiantes como para profesionales.
